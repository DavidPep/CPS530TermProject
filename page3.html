    Backend with Flask

  • Using flask
    • <!-- import flask into your python file--> from flask import Flask app = Flask(__name__) @app.route('/') <!-- this is your route name--> def hello_world(): return 'Hello, World!' <!-- This returns text to creat a html page--> <!-- Using templates to show html--> from flask import render_template <!-- import render template to return a html file--> @app.route('/') @app.route('/home')<!-- you an have multiple routes pointing to one page--> def hello(): return render_template('index.html')<!-- now you can display html code --> <!-- Using python in your html--> {% %} <!-- this is a code block add this to you html files to write python code--> <!--example the below code changes you webpage title--> {% if title %} <title>Flask Site - {{title}} </title> {% else %} <title>Flask Site</title> {% endif %} <!--using a layout and block content to change pages --> First create a layout.html file in that file add all of your css and script tags that will be used in your menus and footerto have the page content always show without writing it over and over you can use block content. <body> {% block content %} {% end block content %} <!--displays your content --> </body> Next create another html file call home.html when done you can inherit layout.html by adding {% extends "layout.html" %} {% block content %} <!-- add you home page body content here--> {% end block content %}
